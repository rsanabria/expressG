# ExpressG
*Version 1.2*
Minimal setup for express apps.

  - [Mongoose](http://mongoosejs.com/)
  - [Cors](https://github.com/expressjs/cors)
# Requirements
-  [Node.js](https://nodejs.org/)
# Installation
```
$ npm install
```
# Usage
- Dev environment
```
$ npm run dev
```
- Production environment
```
$ npm run serve
```
